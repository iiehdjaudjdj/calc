# Calculator Application

## Project Description

This is a modern calculator app made with HTML, CSS, and JavaScript. The calculator has a clean design that works on all screen sizes including mobile phones. It can do basic math like add, subtract, multiply, divide, and modulo. The buttons have loading animations to stop double-clicking when doing calculations.


## Instructions to Run the Project

To run the project, just open the index.html file in your web browser. You can double click the index.html file or open it with your web browser. You do not need a server or to install anything. You can also open it using a local server like XAMPP, WAMP, or Live Server extension in VS Code.

For best experience, use modern web browsers like Chrome, Firefox, Edge, or Safari. The calculator works on all screen sizes so you can use it on desktop, tablet, or mobile phone.
